# Task-5 Ecommerce Product Catalog

A responsive mini e-commerce web application built using HTML, CSS, and JavaScript as part of the Full Stack Web Development Internship Task-5.

## Features

- Product Listing Page
- Dynamic Product Loading using JSON
- Search Products
- Category Filter
- Sorting Options
- Pagination
- Product Detail Page
- Add To Cart
- Update Quantity
- Remove Product from Cart
- Cart Total Calculation
- LocalStorage Integration
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
- JSON

## How To Run The Project

1. Download or clone the repository
2. Open the project folder in VS Code
3. Install the Live Server extension
4. Right click index.html
5. Click "Open With Live Server"
